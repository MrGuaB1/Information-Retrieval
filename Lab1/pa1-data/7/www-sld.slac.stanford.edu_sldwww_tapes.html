access to the sld tape database access to the sld tape database an oracle database is used as the basis of the tape management system for sld the following commands are used to control access to sld tapes and can be used from both the vax and vm machines getfree distributes tapes to users provided the users have been entered in the database as sld personnel makefree will allow the owner of a cartridge to return a cartridge to the pool of free available for use cartridges qtape will return information on tape from the database give will transfer ownership of a tape to another sld user newcom will allow tape_owner to change the comment of the tape mytapes will list out all tapes of a specified user either on screen or in a file default is to list user's tapes tapes are accessed from ida using the opentape command also see the sld user tape policy tony johnson last updated
