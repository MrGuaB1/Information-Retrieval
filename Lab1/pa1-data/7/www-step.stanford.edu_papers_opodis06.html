sanchez sipma gill manna distributed priority inheritance for real time and embedded systems distributed priority inheritance for real time and embedded systems cesar sanchez henny b sipma christopher gill zohar manna we study the problem of priority inversion in distributed real time and embedded systems and propose a solution based on a distributed version of the priority inheritance protocol pip the priority ceiling protocol pcp was designed in the context of centralized systems as a modification of pip that also prevents deadlocks previous approaches to priority inversions in distributed systems use variations of pcp which requires maintaining a global view of the acquired resources involving a high communication overhead this paper presents a distributed pip built on top of a deadlock avoidance schema since the system is already deadlock free and priority inversions can be detected locally we obtain an efficient dynamic resource allocation system that prevents deadlocks and handles priority inversions to appear in opodis 06 henny sipma sipma cs stanford edu
