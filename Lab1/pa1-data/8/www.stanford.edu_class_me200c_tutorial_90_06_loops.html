fortran 90 tutorial 6 loops do loops may have either of the two forms do index variable start end step statements end do or do n index variable start end step statements n continue in both cases the step variable is optional and is assumed to have the value 1 if omitted as with if else and case constructs do loops may also have names while loops are allowed they have the general form do while logical argument statements end do an infinite loop may be performed as follows do conditional statement exit end do in this instance a set of statements are executed repetitively until the conditional statement is satisfied when this occurs program control jumps to the next executable statement following the end do copyright 1996 7 by stanford university all rights reserved
