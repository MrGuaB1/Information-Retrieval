cmt package cmt packages a cmt package represents a set of software components eg applications libraries tools etc and the components themselves are either independent or related to each other the cmt environment provides conventions for naming packages files and directories as well as for addressing them in addition tools are provided for automating the implementation of these conventions cmt permits configuration requirements to be described and automatically deduces from the description the effective set of configuration parameters needed to operate the packages typically for building them or using them last updated by chuck patterson 02 07 2005
