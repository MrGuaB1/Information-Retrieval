cea cea chemical equilibrium with applications cea is a computer program published by nasa glenn with the purpose of calculating chemical equilibrium product concentrations from any set of reactants and determining thermodynamic and transport properties for the product mixture although it is somewhat dated this program is still extremely useful in calculating the performance of rocket motors and engines for more information please check out the cea website at nasa glenn there you can find a web based version of the program as well as the downloadable installation files for running it locally if you are attempting to run cea on an intel based apple computer please see the install guide written by jonah zimmerman home people research courses resources facilities peregrine cea
