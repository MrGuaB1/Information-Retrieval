compilation instructions download jess jar from http herzberg ca sandia gov jess please note that the swrl jess bridge project requires jess version 7.1 p1 or higher put jess jar in your classpath put protege jar and protege owl jar on your classpath these are located in the protege installation directory use ant and build xml to create swrl jess bridge jar it will be placed in the dist subdirectory copy swrl jess bridge jar to the protege owl plugins subdirectory of the protege installation directory
