accessing the space weather prediction center's lists of solar flares the national weather service's space weather prediction center maintains lists of solar flares detected by their goes satellites these lists can be accessed at current list of flares last couple months list of previous flares the current list is readily accessible but the lists of previous flares are compressed and zipped tar gz each directory is archived using the tar utility and the tar file is then compressed using a gnu zip utility these files can be opened using standard unix tar and gzip utilities if you have a windows platform you will need to use winzip or one of the open source free versions such as 7 zip see download 7 zip free
