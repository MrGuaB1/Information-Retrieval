convert to an experiment set an arraylist or result set list is a vehicle for organizing data for your own personal use it is easy to construct on your own computer but cannot be easily shared or published also the information it contains is limited to identifying microarrays and optionally specifying data quality filters for use with those data an experiment set on the other hand can only be constructed using the database's web interface it is designed to be easily shared and published and can contain as much information about an experimental design eg factors variables and descriptions as you choose to specify an experiment set or more than one is required in order to publish data and is also very useful for communicating information about your experiments to collaborators before publication how does the conversion work the first page of the conversion tool allows you to select one of the lists in your loader arraylists directory the second page displays the arrays contained
