the saturn software analysis project home overview publications people download examples related projects contact saturn precise and scalable software analysis examples saturn has a user interface system allowing the facts computed by an analysis to be overlaid on and hyperlinked with the original source code you can use the user interface to browse the databases of facts computed by the analyses we have developed a few notes on using the query page most of the databases have self explanatory names eg linux alias analysis has the results of an alias analysis run on the linux kernel the simplest way to see results is to choose a database set type to keyword and type the name or part of the name of a function in the query box and hit search entering something in the category field restricts results to facts in that category you can learn about what categories of facts are available by browsing some results note that the databases available here are the ones we use for our routine work thus they can be updated or deleted at any time
