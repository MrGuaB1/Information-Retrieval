the process of switching from the majordomo mailing list service to a new system called mailman the new service offers better performance and advanced features like spam filtering list archiving and a web interface by october 2006 it services plans to migrate all of the existing lists on majordomo to the new mailman service there will be a phased migration through october but the mailman program is available now for the creation of new lists you can read about the new service and request a new list via the mailman web site http mailman stanford edu if you would like to convert your existing majordomo mailing list to mailman please submit a helpsu ticket the general functionality for the mailing list servers will remain the same during the transition lists that are converted to or created on mailman will retain the familiar lists stanford edu address the mailman service will make it easier and friendlier to manage your mailing lists please give it a test drive the next time you need a
