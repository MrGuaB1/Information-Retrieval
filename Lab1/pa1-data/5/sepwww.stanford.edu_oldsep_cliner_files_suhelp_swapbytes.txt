swapbytes swap the bytes of various data types swapbytes stdin optional parameters stdout required parameters none optional parameters in float input type float double double short short ushort unsigned short long long ulong unsigned long int int outpar dev tty output parameter file contains the number of values n1 other choices for outpar are dev tty dev stderr or a name of a disk file notes the byte order of the mantissa of binary data values on pc's and dec's is the reverse of so called big endian machines ibm rs6000 sun etc hence the need for byte swapping capability the subroutines in this code have been tested for swapping between pcs and big endian machines but have not been tested for dec products caveat 2 byte short 4 byte long 4 byte float 4 byte int and 8 bit double assumed credits cwp john stockwell jan 1994 institut fur geophysik hamburg jens hartmann supplied byte swapping subroutines
