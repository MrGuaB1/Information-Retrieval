vizster visualizing online social networks vizster visualizing online social networks jeffrey heer danah boyd download vizster you will need to have java installed then simply click the vizster jar file to launch the application vizster does not include any social network data you can either load data from an xml file describing the network or from a custom mysql database to load data from xml click cancel at the initial database login dialog and use the file menu to open a file such as this sample xml file to load data from mysql you will need to create your own mysql installation using this schema then simply enter the database location login and password into vizster's login dialog vizster was built using the alpha release of the prefuse visualization toolkit vizster is open source software released under a bsd license
