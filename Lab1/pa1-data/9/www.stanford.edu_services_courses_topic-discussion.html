support for courses online discussion online discussion courseforum courseforum is an open source discussion tool available through coursework stanford's course management system anyone with access to courseforum can post topics courseforum is ideal for collaborative projects because it supports private forums and forums for specified user groups its search feature allows instructors to find student specific postings that can then be printed for grading courseforum synchronizes with coursework's course enrollment information thereby eliminating the need to add users to the forum manually
