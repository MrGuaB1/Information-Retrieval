clement gerdes kahane lfg02 abstract an lfg type grammar for german based on the topological model lionel clement kim gerdes and sylvain kahane abstract proceedings of lfg02 csli publications on line this paper proposes a description of german word order in a lfg type grammar contrary to earlier lexical functional grammar description of german our grammar uses as c structure the topological model this gives us a simpler grammar which covers partial vp fronting intraposition extraposition auxiliary flip and all other order possibilities for a verbal dependent our grammar is implemented in an lfg parser
