lareau wanner geaf07 abstract towards a generic multilingual dependency grammar for text generation francois lareau and leo wanner abstract for practical multilingual text generation efficient development and representation of large scale grammatical and lexical resources are crucial one way to ensure efficiency is to share resources as much as possible between languages we present some preliminary work on shared grammatical resource development within the framework of the meaning text theory emphasizing the lexicalist point of view we show that rich dictionaries allow for more generic grammar rules which can be used for several languages so that the number of language specific rules is kept low we also discuss the benefits of shifting the workload to the dictionaries from the viewpoint of extension and consistency control as well as the impact it has on the organization of work furthermore we address evaluation methodologies for the shared grammatical resources we develop proceedings of geaf07 csli publications on line proceedings toc proceedings as a single large pdf file
