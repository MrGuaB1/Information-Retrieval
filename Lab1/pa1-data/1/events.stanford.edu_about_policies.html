calendar admin system via sunetid once logged in the calendar administrator can add new events or edit previously submitted events if you represent a stanford academic department organization or student group which does not have a calendar administrator you may complete our online training and request an account contact us via helpsu if you have additional questions about obtaining an administrator account what events can be posted approved events that are open to the stanford community or open to the public can be advertised on the stanford event calendar if they meet the following requirements the event is sponsored by your registered organization the event venue has been reserved for the proposed date and time required approvals are on file eg contracts with speakers or vendors the event is not a party see student activities sal definition of a party outside companies or organizations do not qualify as stanford sponsors and may not post events for more information about the event approval process please refer to the registrar's scheduling office office of special events & protocol and for student events student activities and leadership sal when will events be posted published events will generally be posted within 8 hours of submission the calendar editor reviews newly published events regularly if an event does not appear to meet the requirements it will be removed from the system publication of events is at the discretion of the calendar editor can my event be a featured event the calendar editor selects events to be listed on the featured events page to suggest an event to be featured contact us via helpsu please include the date title and sponsoring organization of your event how do i change event information on the calendar as a calendar administrator you can edit cancel or remove previously entered events a change of venue date time or scope must be approved by the space granting entity if a publicized event has been canceled we strongly recommend that you mark the event as such under the publish tab of the event editing screen rather than removing it from the system how do i add a photo or image to an event listing images should be cropped to 150 x 150 pixels and saved as jpeg on the image tab of the event entry screen you will be prompted to browse for the image file on your computer similar to attaching files in most e mail programs you may also select from a library of generic images available from the image tab how can i integrate event listings into my own website all the listings in the stanford event calendar are available via xml feeds as well as ical and rss feeds the xml feeds provide the richest source of data and the most advanced options for integration into your website please refer to the xml feed documentation for more details and code samples how can i have my event recorded for stanford on itunes u and or youtube the sponsoring department is responsible for producing the audio or video recording we recommend using stanford video for more information on how to contribute your recording to stanford on itunes u or youtube please visit their contributor information site what are the
