topdrawer for win32 this executable td exe runs topdrawer under windows nt 9x the calling sequence is td exe file_name where file_name should be replaced by the actual name of the topdrawer input file currently the program is set to generate a postscript file with the name file_name ps all other output options are disabled topdrawer is a plotting program that was originally developed at slac this version is a greatly enhanced version generated at fermilab the manual can be found here also mirrored locally here the nt version was generated by taking the fermilab source files and compiling them under compaq visual fortran version 6.5 for questions or problems please contact gennady stupakov stupakov slac stanford edu or karl bane kbane slac stanford edu 29 jan 2001
