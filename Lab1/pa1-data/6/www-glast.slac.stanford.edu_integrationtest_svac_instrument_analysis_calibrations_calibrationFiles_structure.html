calibration files directory structure calibration files directory structure the calibration meta database stores pointers to the real calibration files which are stored on disk this page describes the organization of the entries in the meta database and the corresponding directory structure 1 all entries in the calib database will start with latcalibroot followed by the subsystem name ie latcalibroot cal latcalibroot tkr 2 latcalibroot is set by the general glast group setup 3 no other environment variables will be used 4 sas provides an nfs disk to store the calibration files this disk is backed up by sas scs 5 currently latcalibroot points to nfs slac g glast calibrations 6 the disk is also accessible through ftp as ftp ftp glast slac stanford edu glast calibrations
