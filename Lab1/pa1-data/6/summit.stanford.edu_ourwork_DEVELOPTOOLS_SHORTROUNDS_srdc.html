short rounds development consortium home page short rounds software overview short rounds is a case presentation project that allows a student to quickly go through a set of patient files that supports text sound pictures and quicktime movies based on the information presented a student is required to make a differential diagnosis clinical faculty members can enter cases using any word processor and import the cases into short rounds for interactive presentation short rounds is based on pedspresents an earlier case presentation project developed at summit web demo consortium there is a short rounds development consortium srdc for organizations interested in writing clinical cases for short rounds there is a consortium home page which has links to information publications and a membership application ordering and info for more information send mail to shortrounds summit stanford edu development team ramon felciano dr parvati dev rory mather system requirements use of this program requires
