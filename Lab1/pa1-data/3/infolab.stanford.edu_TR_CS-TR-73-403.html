report number cs tr 73 403 institution stanford university department of computer science title hints on programming language design author hoare car date december 1973 abstract this paper based on a keynote address presented at the sigact sigplan symposium on principles of programming languages boston october 1 3 1973 presents the view that a programming language is a tool which should assist the programmer in the most difficult aspects of his art namely program design documentation and debugging it discusses the objective criteria for evaluating a language design and illustrates them by application to language features of both high level languages and machine code programming it concludes with an annotated reading list recommended for all intending language designers ftp reports stanford edu pub cstr reports cs tr 73 403 cs tr 73 403 pdf
