tap lane proxy bookmarklet how does it work a bookmarklet is a combination of a browser bookmark and a javascript application in other words it's a bookmark that can be programmed to do things the lane proxy bookmarklet simply prepends the lane proxy url to the current url in your browser this allows you to view the url as though you were on campus giving you access to materials licensed for use by the stanford academic community if you re not authenticated you ll be prompted for your sunetid and password caveats the lane proxy bookmarklet only works with resources lane has licensed and configured to work with our proxy server if invoked on a resource we do not license and or have not yet configured you will see an error message the lane proxy server cannot complete your request because this host has not been configured properly we make no attempt to configure our proxy server for every resource only those we license and for which a stanford ip address grants access get
