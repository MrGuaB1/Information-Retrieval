report number cs tr 80 827 institution stanford university department of computer science title on the parallel computation for the knapsack problem author yao andrew chi chih date november 1980 abstract we are interested in the complexity of solving the knapsack problem with n input real numbers on a parallel computer with real arithmetic and branching operations a processor time tradeoff constraint is derived in particular it is shown that an exponential number of processors have to be used if the problem is to be solved in time t le sqrt n 2 ftp reports stanford edu pub cstr reports cs tr 80 827 cs tr 80 827 pdf
