report number cs tr 90 1340 institution stanford university department of computer science title programming in qlisp author mason ian a author pehoushek joseph d author talcott carolyn l author weening joseph s date october 1990 abstract qlisp is an extension of common lisp to support parallel programming it was initially designed by john mccarthy and richard gabriel in 1984 since then it has been under development both at stanford university and lucid inc and has been implemented on several commercial shared memory parallel computers qlisp is a queue based shared memory multi processing language this report is a tutorial introduction to the stanford dialect of qlisp ftp reports stanford edu pub cstr reports cs tr 90 1340 cs tr 90 1340 pdf
