report number cs tr 75 500 institution stanford university department of computer science title towards better structured definitions of programming languages author kurki suonio reino date september 1975 abstract the use of abstract syntax and a behavioral model is discussed from the viewpoint of structuring the complexity in definitions of programming languages a formalism for abstract syntax is presented which reflects the possibility of having one defining occurrence and an arbitrary number of applied occurrences of objects attributes can be associated with such a syntax for restricting the set of objects generated and for defining character string representations and semantic interpretations for the objects a system of co operating automata described by another abstract syntax is proposed as a behavioral model for semantic definition ftp reports stanford edu pub cstr reports cs tr 75 500 cs tr 75 500 pdf
