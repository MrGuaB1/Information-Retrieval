of the property p now since atomic encoding formulas are monadic ie they always have a single object term to the left of a 1 place property term we can describe their truth conditions as follows suppose that each 1 place property in the domain of relations receives in addition to an exemplification extension an encoding extension each object that is in the encoding extension of property p represents the fact that the object encodes p but whereas the exemplification extension of a property or relation may vary from world to world let us say that the encoding extension of a property does not vary from world to world indeed the encoding extension of a property is not relativized to a world we can nevertheless define world relative truth conditions for atomic encoding formulas when x if an atomic encoding formula is true at any world w it is true at every world w this fact when expressed from the point of view of our language in which modality is primitive is just our logical axiom for encoding for that axiom asserts that if an atomic encoding formula is possibly true it is necessarily true some consequences logical theorem this theorem asserts an object possibly encodes a property if and only if it necessarily encodes that property clearly the conditional in the left to right direction is true since it is an instance of our logical axiom for encoding the conditional in the right to left direction is true because if it is necessary that x encodes f then it is true simpliciter that x encodes f and so it is possibly true that x encodes f logical theorem this theorem tells us that any property in fact encoded by an object is a property that is essentially encoded by that object and vice versa abstract objects are individuated by the properties they encode these properties are even more essential to their identity than are the properties they may necessarily exemplify for example the property being a detective is a constitutive property of sherlock holmes this is a property holmes encodes it is property we use to specify which abstract object holmes is by contrast the property of not being a building is a property that holmes necessarily exemplifies he shares this property with all other abstract objects though holmes exemplifies this property necessarily it is not one of the properties by which we individuate holmes
