report number cs tr 81 854 institution stanford university department of computer science title on the security of public key protocols author dolev danny author yao andrew c date may 1981 abstract recently the use of public key encryption to provide secure network communication has received considerable attention such public key systems are usually effective against passive eavesdroppers who merely tap the lines and try to decipher the message it has been pointed out however that an improperly designed protocol could be vulnerable to an active saboteur one who may impersonate another user or alter the message being transmitted in this paper we formulate several models in which the security of protocols can be discussed precisely algorithms and characterizations that can be used to determine protocol security in these models will be given ftp reports stanford edu pub cstr reports cs tr 81 854 cs tr 81 854 pdf
