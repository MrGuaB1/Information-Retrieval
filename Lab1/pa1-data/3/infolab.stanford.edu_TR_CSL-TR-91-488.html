report number csl tr 91 488 institution stanford university computer systems laboratory title the stanford ada style checker an application of the anna tools and methodology author walicki michal author skakkebaek jens ulrik author sankar sriram date august 1991 abstract this report describes the ada style checker which was designed and constructed in winter and spring 1989 90 the style checker is based on the stanford anna tools and has been annotated using anna the style checker examines ada programs for correct style which is defined in a style specification language ssl a style checker generator is used to automatically generate a style checker based on a set of style specifications ftp reports stanford edu pub cstr reports csl tr 91 488 csl tr 91 488 pdf
