report number csl tr 92 517 institution stanford university computer systems laboratory title preserving information during online partial evaluation author ruf erik author weise daniel date april 1992 abstract the degree to which a partial evaluator can specialize a source program depends on how accurately the partial evaluator can represent and maintain information about runtime values partial evaluators always lose some accuracy due to their use of finite type systems however existing partial evaluation techniques lose information about runtime values even when their type systems are capable of representing such information this paper describes two sources of such loss in existing specializers solutions for both cases and the implementation of these solutions in our partial evaluation system fuse ftp reports stanford edu pub cstr reports csl tr 92 517 csl tr 92 517 pdf
