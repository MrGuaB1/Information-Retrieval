report number csl tr 79 175 institution stanford university computer systems laboratory title symbolic debugging of optimized code author hennessy john l date july 1979 abstract the long standing conflict between the optimization of code and the ability to symbolically debug the code is examined the effects of local and global optimizations on the variables of a program are categorized and models for representing the effect of optimizations are given these models are used by algorithms which determine the subset of variables whose values do not correspond to those in the original program algorithms for restoring these variables to their correct values are also developed empirical results from the application of these algorithms to local optimization are presented ftp reports stanford edu pub cstr reports csl tr 79 175 csl tr 79 175 pdf
