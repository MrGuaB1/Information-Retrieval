report number csl tr 94 654 institution stanford university computer systems laboratory title multipliers and datapaths author al twaijry hesham author flynn michael j date december 1994 abstract people traditionally have considered the number of counters in the critical path as the metric for the performance of a multiplier this report presents the view that tree topologies which have the least number of levels do not always give the fastest possible multiplier when constrained to be part of a microprocessor it proposes two new topologies hybrid structure and higher order arrays which are faster than conventional tree topologies for typical datapaths ftp reports stanford edu pub cstr reports csl tr 94 654 csl tr 94 654 pdf
