pollack lab homepage software dr integrator dna rna integrator dr integrator is a freely available statistical software tool to perform integrative analyses on paired dna copy number and gene expression data dr integrator identifies genes with significant correlations between dna copy number and gene expression and also performs a supervised analysis that captures genes with significant alterations in both dna copy number and gene expression between two sample classes dr integrator is available as a plug in application to microsoft excel and as a package for the r statistical computing environment when publishing using dr integrator please cite the following reference dr integrator a new analytic tool for integrating dna copy number and gene expression data salari k tibshirani r pollack jr bioinformatics 2009 dec 22 epub ahead of print ms excel plug in r package user manual
