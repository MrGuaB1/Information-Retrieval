how it works basic how it works basic 11 26 2006 fundamental law light travels in a straight line a candle emitting light behaves in this way tracing out some of the light rays illustrates this point adding a light tight box blocks light from entering box if we add a small hole at the center some light can enter we can trace light reflecting off of the rest of the candle in the same way and form an image of the candle upsided down at the back of the box if we now add film at the back of the box a cap at the hole to control when light enters the box and exposes the light sensitive film we have a camera that's it this is a basic pinhole camera and all that is needed to make images the rest is refinement to a specific end
