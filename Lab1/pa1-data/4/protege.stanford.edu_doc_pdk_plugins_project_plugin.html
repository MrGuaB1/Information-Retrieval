tab widget how to write a project plug in a project plug in is a plug in whose methods are called at various points in a protege project's life cycle this plug in type allows you to manipulate a project and the project ui as you would like the interface plugin projectplugin shows the methods that are available for you to implement the easiest way to develop such a plug in is to subclass abstractprojectplugin implement the abstract methods if you only want to implement one or two of the methods then you may choose to subclass projectpluginadapter this class implements all of the required methods with do nothing implementations thus you can just override the method s that you want we have provided example code for a project plug in that just adds a menu item to the edit menu it also prints to the console whenever any of its methods are called follow these steps to compile and run the project plug in example download the source code please make sure to preserve the path
